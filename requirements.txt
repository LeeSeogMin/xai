# IMPORTANT: Use Python 3.11 (tested: 3.11.x). The original experiments were run
# on Windows; this exact stack was re-verified end-to-end on macOS (Apple Silicon).

# Why pinned: with unpinned (>=) ranges a fresh install on Python 3.14 resolves to
# numpy 2.x / torch 2.12 / xgboost 3.2, which segfaults on macOS due to a duplicate
# OpenMP runtime (torch and xgboost each load their own libomp). Pinning the stack
# below avoids that and makes results reproducible across Mac and Windows.
#   pip install -r requirements.txt
# =============================================================================
